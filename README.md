This is a simple weather app that was created as a portfolio project to express knowledge in a specific technology stack.

&nbsp;


## Usage/Examples

```javascript
REACT_APP_OPEN_WEATHER_API_KEY = " INSERT OPEN WEATHER MAP API KEY " 
```
